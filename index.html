Latest Updates

24/02/15 - Added the conditional tag to the head. This fixes email rendering on Windows Phones. Also allows Outlook 2003 possibly higher to render css3 properties, e.g. media queries, border-radius, box-shadow.

03/02/15 - Added the Outlook break & Outlook table alignment issues fix. Wraps your left and right aligned tables in two cells. Makes the need for a 20px gap redundant. Make sure to amend the width of the table in the commented out code, to match the width of your wrapper table

04/09/14 - Yahoo/Gmail app "centering fix" added for centered tables being left aligned when a tables size is set smaller than full width to add a left and right margin. A full width container table has been added with align="center" on the cell, then align left is added to the nested tables content cell for left aligned the content.
